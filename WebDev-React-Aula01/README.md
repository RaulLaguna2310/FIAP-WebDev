## Clonando o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/nome-do-repositorio.git

# Acesse a pasta do projeto
cd nome-do-repositorio
```

## Acessando e Inicializando o Projeto

```bash
# Entre na pasta do projeto
cd meu-app

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
